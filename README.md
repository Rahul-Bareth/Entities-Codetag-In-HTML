# Entities & Code Tag in HTML

This project demonstrates the use of HTML entities and the `<code>` tag. It covers how to display reserved HTML characters using entities such as `&lt;`, `&gt;`, and `&copy;`, preserve formatting with the `<pre>` tag, and present source code snippets using the `<code>` element. This practice project helps beginners understand how to display special characters and formatted code correctly in web pages.
